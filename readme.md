# Vault Obsidian

Notes, projets, études, médias, contacts.

## Structure

- `Daily-notes/` — journal quotidien
- `clippings/` — articles et liens capturés
- `Categories/` — MOC (Maps of Content) pour naviguer
- `Bases/` — vues globales de toutes les notes
- `Template/` — templates

## Templates disponibles

| Template | Usage |
| -------- | ----- |
| `Note-Template` | Note générale ou atomique |
| `Daily-Note-Template` | Note quotidienne |
| `UE-Template` | Wrapper d'une matière |
| `UE-Note-Template` | Note de cours |
| `Projects-Template` | Fiche projet |
| `Project-Note-Template` | Note liée à un projet |
| `Person-Template` | Fiche contact personne |
| `Organisation-Template` | Fiche contact organisation |
| `Clippings-template` | Capture web |
| `Book-Template` | Livre |
| `Show-Template` | Film ou série |
| `VideoGame-Template` | Jeu vidéo |
